# DummyJSON APP Integrative Programming

This is an integrative programming project developed by Mharx Nevin and Francine Nicole.

The system is a web-based application built using PHP and MySQL, and is run locally using XAMPP.

The project demonstrates core concepts in full-stack development, including user authentication, database management, and session handling. It features several functional modules such as user registration and login, a dashboard interface, product and post management, and a shopping cart system.
