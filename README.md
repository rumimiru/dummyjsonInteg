# DummyJSON APP Integrative Programming

This is an integrative programming project developed by Mharx Nevin and Francine Nicole. The system is a web-based application built using PHP, designed to demonstrate core concepts in full-stack development, including user authentication, product management, and session handling.

The project features multiple functional modules such as user registration and login, a dashboard interface, product and post management, and a shopping cart system. It follows a structured file organization with separate directories for assets, configuration, images, and reusable components to ensure maintainability and scalability.

Overall, this project showcases practical implementation of web development principles, combining both front-end and back-end logic into a cohesive and functional application.
